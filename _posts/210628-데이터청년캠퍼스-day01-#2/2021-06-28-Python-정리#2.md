## immutable data 와 mutable data
```python
'''
immutable data
'''
a = b = 1   # a = 1, b = 1
b += 1      # a = 1, b = 2

'''
mutable data
'''
a = b = [1,2,3] # a = [1,2,3],   b = [1,2,3]
b.append(4)     # a = [1,2,3,4], b = [1,2,3,4]
```

- 자기 자신이 바뀔 수 있는 **container**이면 mutable하다고 한다.
